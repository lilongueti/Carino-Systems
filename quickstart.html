What is this website?

It is meant to aid me supporting people who require my services. Having quick access to resources and platforms I need or enjoy.

OS images

A section to get ISO files of many useful operating systems, biased towards x86_64 use cases.

What are the profiles for:

FOSS profile delivers the most free and open software for basic, gaming, corporate and IT use cases, this is a profile for FOSS suporters who are going to be tweaking and doing stuff manually anyway.

Basic profile delivers the most basic software for media playback, internet browsing, office suite, file manipulation, communication and remote assistance. Adding the Gaming packages will install popular gaming platforms and utilities.

Corporate profile delivers the most packages for office work, including applications for specific working ecosystems like Microsoft's and Google's.

IT profile delivers the most packages for networking, remote desktop, some coding and some office work. It enables Hyper-V and other features neccesary for running VMs on Windows, or install the @virtualization bundle plus other packages necessary to start creating VMs right away on Linux Systems. It ain't a developer profile though.

Linux systems come with preinstalled AMD drivers, while on Windows you have to install them manually, drivers are here.

NVIDIA drivers will be installed through all of the scripts if an NVIDIA card is detected in the system.
