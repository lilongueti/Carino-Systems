What are the profiles for:

Basic profile delivers the most basic software for media playback, internet browsing, office suite, file manipulation, communication and remote assistance.

Work profile delivers the basic profile plus the most packages for office work, including applications for specific working ecosystems like Microsoft's and Google's.

Gaming profile is just basic profile plus Steam, Lutris and a bunch of utilities.

IT profile delivers the most packages for networking, remote desktop, some coding and some office work. It enables Hyper-V and other features neccesary for running VMs on Windows, or install the @virtualization bundle plus other packages necessary to start creating VMs right away on Linux Systems. It ain't a developer profile though.

AMD drivers on Windows must be installed manually, drivers are here. AMD drivers come preinstalled on Linux Systems, no need to look for them in any script.

NVIDIA drivers will be installed through all of the scripts if an NVIDIA card is detected in the system.
