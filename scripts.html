What are the profiles for:

Basic profile delivers the most basic software for media playback, internet browsing, office suite, file manipulation, communication and remote assistance. Adding the Gaming packages will install popular gaming platforms and utilities.

Corporate profile delivers the most packages for office work, videocalls, including applications for specific working ecosystems like Microsoft's and Google's. Installing the IT packages delivers the most solutions for networking, remote desktop, some coding and some office work to deal with others troubleshooting. It enables Hyper-V and other features neccesary for running VMs on Windows, or install virtualization solutions to start creating VMs or containers right away on Linux Systems. It ain't a developer profile though.

FOSSprofile delivers the most free and open software packages for basic, gaming, work and IT use cases, this is a profile for FOSS suporters who are going to be tweaking, criticizing and doing stuff manually anyway.

Linux systems come with preinstalled AMD drivers, while on Windows you have to install them manually, drivers are here.

NVIDIA drivers will be installed through all of the scripts if an NVIDIA card is detected in the system.
